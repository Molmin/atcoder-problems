配点 : $100$ 点

## 問題文

整数 $R,C$ と $2$ 行 $2$ 列からなる行列 $A$ が与えられるので、 $A_{R,C}$ を出力してください。

## 制約

- 入力は全て整数
- $1 \le R,C \le 2$
- $0 \le A_{i,j} \le 100$

## 入力

入力は以下の形式で標準入力から与えられる。

> $R$ $C$
> 
> $A_{1,1}$ $A_{1,2}$
> 
> $A_{2,1}$ $A_{2,2}$

## 出力

答えを整数として出力せよ。

```input1
1 2
1 0
0 1
```

```output1
0
```

$A_{1,2}=0$ です。

```input2
2 2
1 2
3 4
```

```output2
4
```

$A_{2,2}=4$ です。

```input3
2 1
90 80
70 60
```

```output3
70
```

$A_{2,1}=70$ です。
