## Problem Statement

Compute $A \times B$.

## Constraints

- $1 \leq A \leq 100$
- $1 \leq B \leq 100$
- All values in input are integers.

## Input

Input is given from Standard Input in the following format:

> $A$ $B$

## Output

Print the value $A \times B$ as an integer.

```input1
2 5
```

```output1
10
```

We have $2 \times 5 = 10$.

```input2
100 100
```

```output2
10000
```
