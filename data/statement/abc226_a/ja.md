配点 : $100$ 点

## 問題文

小数第三位までで表すことのできる実数 $X$ が、小数第三位まで入力されます。<br>
$X$ を小数第一位で四捨五入した結果として得られる整数を出力してください。

## 制約

- $0 \leq X &lt; 100$
- $X$ は小数第三位までで表現可能である。
- $X$ は小数第三位まで与えられる。

## 入力

入力は以下の形式で標準入力から与えられる。

> $X$

## 出力

$X$ を小数第一位で四捨五入して得られる整数を出力せよ。

```input1
3.456
```

```output1
3
```

$3.456$ の小数第一位は $4$ であるので、$3.456$ を小数第一位で四捨五入した値は $3$ となります。

```input2
99.500
```

```output2
100
```

```input3
0.000
```

```output3
0
```
