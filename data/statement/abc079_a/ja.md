配点 : $100$ 点

## 問題文

$1118$ のような、$3$ つ以上の同じ数字が連続して並んだ $4$ 桁の整数を **良い整数** とします。

$4$ 桁の整数 $N$ が与えられるので、$N$ が **良い整数** かどうかを答えてください。

## 制約

- $1000 \leq N \leq 9999$
- 入力は整数からなる

## 入力

入力は以下の形式で標準入力から与えられる。

> $N$

## 出力

$N$ が **良い整数** ならば `Yes` を、そうでなければ `No` を出力せよ。

```input1
1118
```

```output1
Yes
```

$1$ が $3$ つ連続して並んでいるので **良い整数** です。

```input2
7777
```

```output2
Yes
```

全ての数字が同じ場合も **良い整数** になります。

```input3
1234
```

```output3
No
```
