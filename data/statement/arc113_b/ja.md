## 問題文

正の整数 $A,B,C$ が与えられます。$A^{B^C}$ の $10$ 進法での $1$ の位を求めてください。

## 制約

- $1\leq A,B,C \leq 10^9$
- $A,B,C$ は整数である

## 入力

入力は以下の形式で標準入力から与えられる。

> $A$ $B$ $C$

## 出力

$A^{B^C}$ の $10$ 進法での $1$ の位を出力せよ。

```input1
4 3 2
```

```output1
4
```

$4^{3^2}=4^9=262144$ の $1$ の位は $4$ です。

```input2
1 2 3
```

```output2
1
```

```input3
3141592 6535897 9323846
```

```output3
2
```
