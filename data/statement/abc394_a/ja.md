配点 : $100$ 点

## 問題文

数字からなる文字列 $S$ が与えられます。

$S$ から `2` 以外の文字を削除し、残った文字を順序を保って結合した文字列を求めてください。

## 制約

- $S$ は数字からなる長さ $1$ 以上 $100$ 以下の文字列
- $S$ は `2` を $1$ つ以上含む

## 入力

入力は以下の形式で標準入力から与えられる。

> $S$

## 出力

答えを出力せよ。

```input1
20250222
```

```output1
22222
```

`20250222` から `0`, `5`, `0` を削除し、残った文字を順序を保って結合することで文字列 `22222` が得られます。

```input2
2
```

```output2
2
```

```input3
22222000111222222
```

```output3
22222222222
```
