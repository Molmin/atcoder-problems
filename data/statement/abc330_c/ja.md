## 問題文

正整数 $D$ が与えられます。

非負整数 $x,y$ に対する $|x^2+y^2-D|$ の最小値を求めてください。

## 制約

- $1\leq D \leq 2\times 10^{12}$
- 入力は全て整数

## 入力

入力は以下の形式で標準入力から与えられる。

> $D$

## 出力

答えを出力せよ。

```input1
21
```

```output1
1
```

$x=4,y=2$ のとき $|x^2+y^2-D| = |16+4-21|=1$ となります。

$|x^2+y^2-D|=0$ を満たすような非負整数 $x,y$ は存在しないので、答えは $1$ です。

```input2
998244353
```

```output2
0
```
