配点 : $500$ 点

## 問題文

$1$ 以上 $N$ 以下の正整数 $x$ であって、ある正整数 $a$ と **$2$ 以上の** 正整数 $b$ を用いて $x=a^b$ と表現できるものはいくつありますか？

## 制約

- 入力は全て整数
- $1 \le N \le 10^{18}$

## 入力

入力は以下の形式で標準入力から与えられる。

> $N$

## 出力

答えを整数として出力せよ。

```input1
99
```

```output1
12
```

問題文中の条件を満たす整数は $1,4,8,9,16,25,27,32,36,49,64,81$ の $12$ 個です。

```input2
1000000000000000000
```

```output2
1001003332
```
