配点 : $100$ 点

## 問題文

文字列 `atcoder` の $L$ 文字目から $R$ 文字目までを出力してください。

## 制約

- $L,R$ は整数
- $1 \le L \le R \le 7$

## 入力

入力は以下の形式で標準入力から与えられる。

> $L$ $R$

## 出力

答えを出力せよ。

```input1
3 6
```

```output1
code
```

`atcoder` の $3$ 文字目から $6$ 文字目までを出力すると `code` となります。

```input2
4 4
```

```output2
o
```

```input3
1 7
```

```output3
atcoder
```
