配点 : $100$ 点

## 問題文

次の長さ $32$ の数列の $K$ 番目の項を出力してください。

```plain
1, 1, 1, 2, 1, 2, 1, 5, 2, 2, 1, 5, 1, 2, 1, 14, 1, 5, 1, 5, 2, 2, 1, 15, 2, 2, 5, 4, 1, 4, 1, 51
```

## 制約

- $1 \leq K \leq 32$
- 入力は全て整数である。

## 入力

入力は以下の形式で標準入力から与えられる。

> $K$

## 出力

$K$ 番目の項を出力せよ。

```input1
6
```

```output1
2
```

$6$ 番目の項は $2$ です。

```input2
27
```

```output2
5
```

$27$ 番目の項は $5$ です。
