配点 : $100$ 点

## 問題文

$N$ 個の互いに区別できないお菓子を、A君とB君で分け合います。
両者とも $1$ 個以上の整数個のお菓子を得るような分け方は何通りありますか？

## 制約

- $N$ は整数
- $1 \leq N \leq 15$

## 入力

入力は以下の形式で標準入力から与えられる。

> $N$

## 出力

答えを整数として出力せよ。

```input1
2
```

```output1
1
```

A君が $1$ 個、B君が $1$ 個取る方法のみ存在します。

```input2
1
```

```output2
0
```

```input3
3
```

```output3
2
```
