配点 : $100$ 点

## 問題文

整数 $a, b, c, d$ が与えられます。<br>
$a \leq x \leq b,\ c \leq y \leq d$ となるように整数 $x, y$ を選ぶとき、 $x - y$ の最大値を求めてください。

## 制約

- 入力は全て整数
- $-100 \leq a \leq b \leq 100$
- $-100 \leq c \leq d \leq 100$

## 入力

入力は以下の形式で標準入力から与えられる。

> $a$ $b$
> 
> $c$ $d$

## 出力

答えを出力せよ。

```input1
0 10
0 10
```

```output1
10
```

$(x, y) = (10, 0)$ のとき最大値 $x - y = 10$ をとります。

```input2
-100 -100
100 100
```

```output2
-200
```

```input3
-100 100
-100 100
```

```output3
200
```
