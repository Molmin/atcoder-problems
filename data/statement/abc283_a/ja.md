配点 : $100$ 点

## 問題文

整数 $A, B$ が与えられます。
$A^B$ の値を出力してください。

## 制約

- $1 \leq A, B \leq 9$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

> $A$ $B$

## 出力

答えを出力せよ。

```input1
4 3
```

```output1
64
```

$4^3 = 64$ であるので、$64$ を出力します。

```input2
5 5
```

```output2
3125
```

```input3
8 1
```

```output3
8
```
