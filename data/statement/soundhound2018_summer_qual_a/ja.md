配点 : $100$ 点

## 問題文

$2$ つの 整数 $a,b$ が与えられます。
$a+b=15$ であるか、$a\times b=15$ であるか、そのどちらでもないかを判定してください。

ただし、$a+b=15$ と $a\times b=15$ が同時に成り立つことはありません。

## 制約

- $1 \leq a,b \leq 15$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

> $a$ $b$

## 出力

$a+b=15$ であれば `+` を、
$a\times b=15$ であれば `*` を、
どちらでもなければ `x` を出力してください。

```input1
4 11
```

```output1
+
```

$4+11=15$ です。

```input2
3 5
```

```output2
*
```

$3\times 5=15$ です。

```input3
1 1
```

```output3
x
```

$1+1=2$ であり、 $1\times 1=1$であるため、どちらも $15$ ではありません。
