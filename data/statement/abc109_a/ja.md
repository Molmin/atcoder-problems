配点 : $100$ 点

## 問題文

$1$ 以上 $3$ 以下の整数 $A, B$ が与えられます。

$A \times B \times C$ が奇数となるような $1$ 以上 $3$ 以下の整数 $C$ が存在するか判定してください。

## 制約

- 入力はすべて整数である
- $1 \leq A, B \leq 3$

## 入力

入力は以下の形式で標準入力から与えられる。

> $A$ $B$

## 出力

条件を満たすような $C$ が存在するなら `Yes`、そうでないなら `No` を出力せよ。

```input1
3 1
```

```output1
Yes
```

$C = 3$ とすると $A \times B \times C = 3 \times 1 \times 3 = 9$ となり、奇数となります。

```input2
1 2
```

```output2
No
```

```input3
2 2
```

```output3
No
```
