配点 : $100$ 点

## 問題文

相違なる整数 $A$, $B$ があります。

$|A - K| = |B - K|$ となるような整数 $K$ を出力してください。

そのような整数が存在しなければ、代わりに `IMPOSSIBLE` を出力してください。

## 制約

- 入力は全て整数である。
- $0 \leq A,\ B \leq 10^9$
- $A$ と $B$ は相違なる。

## 入力

入力は以下の形式で標準入力から与えられる。

> $A$ $B$

## 出力

条件を満たす整数 $K$ を出力せよ。

そのような整数が存在しなければ、代わりに `IMPOSSIBLE` を出力せよ。

```input1
2 16
```

```output1
9
```

$|2 - 9| = 7$, $|16 - 9| = 7$ であるため、$9$ は条件を満たす整数です。

```input2
0 3
```

```output2
IMPOSSIBLE
```

条件を満たす整数は存在しません。

```input3
998244353 99824435
```

```output3
549034394
```
