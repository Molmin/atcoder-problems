配点 : $400$ 点

## 問題文

整数 $N,K$ が与えられます．
$4$ つの整数の組 $(a,b,c,d)$ であって，以下の条件を両方満たすものは何個あるでしょうか．

- $1 \leq a,b,c,d \leq N$
- $a+b-c-d=K$

## 制約

- $1 \leq N \leq 10^5$
- $-2(N-1) \leq K \leq 2(N-1)$
- 入力される数はすべて整数．

## 入力

入力は以下の形式で標準入力から与えられる．

> $N$ $K$

## 出力

答えを出力せよ．

```input1
2 1
```

```output1
4
```

以下の $4$ 通りです．

- $(a,b,c,d)=(2,1,1,1)$
- $(a,b,c,d)=(1,2,1,1)$
- $(a,b,c,d)=(2,2,2,1)$
- $(a,b,c,d)=(2,2,1,2)$

```input2
2525 -425
```

```output2
10314607400
```
