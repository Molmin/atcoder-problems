配点 : $400$ 点

## 問題文

整数からなる公差 $1$ の等差数列のうち、総和が $N$ であるものはいくつあるでしょうか？  

## 制約

- $1 \leq N \leq 10^{12}$
- $N$ は整数

## 入力

入力は以下の形式で標準入力から与えられる。

> $N$

## 出力

答えを出力せよ。

```input1
12
```

```output1
4
```

- $[12]$
- $[3, 4, 5]$
- $[-2, -1, 0, 1, 2, 3, 4, 5]$
- $[-11, -10, -9, \dots, 10, 11, 12]$

の $4$ 個です。

```input2
1
```

```output2
2
```

- $[1]$
- $[0, 1]$

の $2$ 個です。

```input3
963761198400
```

```output3
1920
```
